Harmonic Extension from Live Particles

Draw a domain, place the point x, choose boundary data f, then click anywhere inside the shape to shoot particles from x. Each Brownian particle hops from x until it hits the boundary; averaging f over where they land estimates the harmonic extension Hf(x) — the solution of Laplace's equation with boundary values f. Outer band = value of f (darker = larger); inner band = empirical hitting distribution (240 bins).

Probabilistic Dirichlet solver: H_f(x) = E_x[ f(B_τ) ] = ∫ f dω(x,·), where ω is harmonic measure (the Brownian exit distribution). For f = x or f = y the answer is exact and known — coordinate functions are harmonic, so H_f(x) equals that coordinate of x. Turbo uses walk-on-spheres, exact for any domain. Drag the black point to move x; press Draw domain then drag to sketch a new shape.
